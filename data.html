Guide

How this conversion works

This converter uses the binary standard, where 1 KB = 1,024 bytes, matching how Windows, macOS and most operating systems report file and drive sizes. This differs from the decimal standard (1 KB = 1,000 bytes) that drive manufacturers often use on packaging, which is one reason a drive can show less capacity than advertised once it's formatted.

A bit is the smallest unit (a single 0 or 1); 8 bits make a byte, which is roughly enough to represent one character of text.

FAQ

Frequently asked questions

Why does my 1TB drive show less than 1,000 GB in my OS?

Manufacturers typically advertise capacity using the decimal standard (1 TB = 1,000,000,000,000 bytes), while your operating system reports it using the binary standard (1 TB = 1,024⁴ bytes). The drive isn't missing space — the two standards are just counting differently.

What's the difference between a bit and a byte?

A bit is a single binary digit (0 or 1). A byte is 8 bits, and is the typical unit for measuring file sizes, memory and storage — internet connection speeds, by contrast, are usually quoted in bits per second (Mbps), not bytes.
